Themes & Custom Theming

Four built-in themes (Raccoon · Material · Fluent · Tabler) with light/dark variants. Use themeVars to override any CSS token, or combine it with a base theme to extend it.

Usage examples
// Extend an existing theme — just override what you want
const grid = new RaccoonGrid({
  theme: 'raccoon',
  themeVars: {
    '--rt-color-primary':       '#27ae60',
    '--rt-color-primary-hover': '#219a52',
    '--rt-color-bg-selected':   'rgba(39, 174, 96, 0.12)',
    '--rt-cell-selected-bg':    'rgba(39, 174, 96, 0.14)',
    '--rt-cell-selected-border':'rgba(39, 174, 96, 0.50)',
  },
  columns: [...],
  data: [...],
});

// Change vars at runtime (theme stays the same)
grid.setThemeVars({
  '--rt-color-primary': '#e74c3c',
});

// Reset all overrides
grid.setThemeVars(undefined);

// Fully custom theme (no base theme — set everything)
const grid2 = new RaccoonGrid({
  // no `theme:` — starts from Raccoon tokens
  themeVars: {
    '--rt-font-family':    '"JetBrains Mono", monospace',
    '--rt-color-bg':       '#0d1117',
    '--rt-color-bg-alt':   '#161b22',
    '--rt-color-bg-header':'#21262d',
    '--rt-color-border':   '#30363d',
    '--rt-color-text':     '#e6edf3',
    '--rt-color-primary':  '#58a6ff',
    // ... set remaining tokens as needed
  },
  columns: [...],
  data: [...],
});
