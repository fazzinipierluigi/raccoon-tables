Flex Columns — proportional width distribution

Action Column — cell overflow & dropdown menu

cellOverflow: true lets a cell visually exceed its boundaries. The action button opens a dropdown menu appended to document.body — use this pattern for menus inside cells to avoid clipping by the scroll container.

Boolean Filter & Lookup (Dropdown) Filter

Active: boolean filter shows a select (All / True / False / Null). Department: lookup filter with static options list — always exact-match (==). Contract: lookup filter that would normally load via AJAX (using static data here for the demo).
