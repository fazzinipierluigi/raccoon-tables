Server-Side Mode

Every sort, filter, page change fires an AJAX request. Requests are debounced and in-flight requests are cancelled (AbortController) on new interaction.

This demo intercepts fetch() and simulates a server with 10,000 rows. The Request Log below shows every request body sent to the server. Latency is simulated at 300ms. Try sorting, filtering, and changing pages.
