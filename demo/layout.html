Layout Save & Restore

grid.getLayout() captures columns (order, visibility, width, pin), sort, filters, pagination and row groups into a plain JSON object. grid.setLayout(layout) fully restores it — useful for persisting personal views server-side.

Try it out

Sort columns, apply filters, pin some columns via the ⋮ header menu, resize columns, then click Save Layout. Change the grid state and click Restore Layout to bring it back. The saved JSON is shown on the right.
