Inline Editing

Double-click an editable cell to open the inline editor. Tab / Shift+Tab navigates between editable cells in the same row. Custom editor component example on the Rating column.

Keyboard shortcuts (with cellSelection enabled): Ctrl+C = copy selection · Ctrl+V = paste · Delete = clear selected cells · Any printable key on an editable cell = start editing
